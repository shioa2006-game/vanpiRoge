# GRID ROGUE: Tactical Roguelite

ターン制ローグライクの思考性と、Survivors系のビルド中毒性を融合したブラウザ向け2Dゲーム。

## 遊び方

`index.html` をブラウザで直接開くだけで起動します。外部依存・サーバー不要。

### キー操作

| キー | 操作 |
|------|------|
| WASD / 矢印キー | 移動 |
| Space | 待機（1ターン経過） |
| E | 宝箱を開ける / アイテム装備 |
| Esc | 拠点に戻る |
| 1 / 2 / 3 | レベルアップ時にスキル選択 |
| Enter | ランを開始 / リザルト画面から戻る |

---

## ゲームシステム

### コアループ

- プレイヤーが**1歩動く or 待機するたびに1ターン**が進む
- プレイヤーが止まっている間は時間が進まない（完全ターン制）
- 敵・弾・クールダウン・状態異常はすべてプレイヤー行動に連動して更新される

### 戦闘

- **近接攻撃**: 敵のいるマスへ移動しようとすると自動発動
- **スキル**: 一定ターンのクールダウン後に自動発動
- プレイヤーは「位置取り」「ルート選択」「敵の誘導」を考えることが主な操作

### マップ

- **手続き生成**のグリッドダンジョン（42×30マス）
- 複数の部屋が通路でつながれた構造
- 視点はプレイヤー追従のスクロール（16×12マスのビューポート）
- フロアを進む条件: **敵を全滅**させると階段が使用可能になる

### フロア構成

| フロア | 内容 |
|--------|------|
| 1〜3 | 雑魚敵ラッシュ（フロアが進むにつれ敵数・強さ増加） |
| 4 | ボスフロア |

## ビルド構築

レベルアップ時に3択からスキルを選択。同じスキルを重ねると強化（最大Lv3）。

**シナジー例:**
- 毒クラウド × ヴォイドブレード装備 → 毒敵に2倍ダメージ
- フロストリング × チェーンライトニング → 凍結中に高連鎖
- アルカンエコー × 高CDスキル → 次スキル2連発
- ウォークライ × ブレードスイープ → バフ中に大範囲一掃
- アイアンスキン × バーサーカーバンド装備 → 瀕死状態で超火力

## ローグライト進行（拠点）

ラン終了後、以下が引き継がれる:

- **ゴールド**（モードにより持ち帰り率が変化）
- **装備品**（ラン中に装備したものをそのまま持ち帰れる）

### 恒久アップグレード

拠点でゴールドを消費して購入。ラン間をまたいで永続。

| 名前 | 効果 | 最大Lv |
|------|------|--------|
| 強靭な体 | 最大HP+15 | 5 |
| 武闘訓練 | 基礎ATK+3 | 5 |
| 防衛訓練 | 基礎DEF+2 | 5 |
| 資金準備 | 開始ゴールド+15 | 3 |
| 熟練の技 | 全スキルCD-1 | 3 |

## セーブデータ

`localStorage` に自動保存（キー: `gr_v1`）。

保存内容:
- ゴールド残高
- 装備インベントリ（4スロット）
- 恒久アップグレードレベル
- 総ラン数・総撃破数

バージョン管理あり。データ破損時は自動で初期化。

## 既知の制限

- 手続き生成の都合で、稀に部屋が孤立する場合がある
- 敵AIはグリーディ移動のため、壁沿いの経路が最短でないケースがある
- スキルはすべて自動発動（プレイヤーが方向・タイミングを指定できない）
- モバイル・タッチ操作非対応
